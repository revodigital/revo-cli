# Revo CLI
<Warning>Questa documentazione potrebbe risultare incompleta e imprecisa in quanto ancora in fase di creazione.</Warning>

## Introduzione
Questa è la linea di comando dell'organizzazione Revo Digital e servirà per poter permettere la creazione dinamica di progetti FrontEnd, BackEnd e MobileApp strutturati secondo determinati pattern e con l'utilizzo di determinate librerie, così da rendere i progetti ready to use e non spendere i primi giorni/settimane sulla configurazione degli stessi.

----

## Installazione
```bash
npm install -g @revodigital/revo-cli
```
----
